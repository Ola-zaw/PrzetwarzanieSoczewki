## Rozpoznawanie człowieka na podstawie obrazu tęczówki

Celem projektu była realizacja aplikacji okienkowej przeznaczonej do biometrycznego rozpozna-
wania osób na podstawie obrazu tęczówki oka. System umożliwia pełną, interaktywną wizuali-
zację procesu przetwarzania, obejmującą segmentację źrenicy i tęczówki, normalizację obrazu do
postaci prostokątnej zgodnie z algorytmem Daugmana oraz ekstrakcję cech przy użyciu falek
Gabora. Aplikacja pozwala na wygenerowanie unikalnego kodu tęczówki oraz jego weryfikację
poprzez obliczenie odległości Hamminga względem innych wzorców.
